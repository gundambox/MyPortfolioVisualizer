## 從 1997-07-03 開始模擬

![1997投入100萬](1997.png)

### 回測資料解讀

1. 美股沒有受到亞洲金融風暴的影響，所以SSO可以先累積資本來應付網路泡沫、次貸風爆跟武漢肺炎
2. SSO 有足夠資本承受下跌
3. 即使台股正2投入後馬上遇到亞洲金融風暴跟網路泡沫而下跌94%，仍然在27年後完勝大盤近2倍
4. 相對健康的美股也是正2完勝原型2倍
5. 光是拿20%到美股就可以避險
6. 台股正2如果規模不夠有下市風險
